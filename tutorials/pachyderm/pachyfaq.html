FAQ

Pachyderm is giving an error/won't let me access a screen
Don't panic. First, try logging out of Pachyderm, then log back in. If you can't log out, open Pachyderm in an incognito/private window, and log in again. Try going to the same screen that gave you the error. If it works just fine, it means your Pachyderm login had timed out. If it still gives you the same error, email John Mehr (jmehr@umn.edu) and ask him to restart the Pachyderm server. We haven't been able to figure out what this problem is, but turning things off and back on again solves a lot of stuff.
What is a tag?
A tag is an HTML element, and is enclosed in angle brackets (<>). These are what tell Pachyderm what to do with the text (or other tags) inside the tag. Almost every tag must be closed with a closing tag, which looks something like </>. Learn more
How do I know when to close which tags?
This will become more intuitive with practice. Try thinking of it this way: the last tag you opened is the first tag you have to close. The exceptions are tags which don't contain any text, like <ht>. Learn more
What do you mean, "I've changed the rendered example"
When I've said this in the manual, I mean that I've added some inline CSS with a style attribute to alter the way the HTML looks. If you're feeling adventurous, take a look at the HTML code for the manual to see exactly what I wrote. Specifically, I specified the size of the image in the Image entry and I changed the size of and added a border around the text in the Horizontal Rule entry. Learn more
Why do we use strong instead of b and em instead of i?
The answer is accessibility. When someone who has poor vision or is blind uses and HTML document, they are more than likely to use a screen reader. This is a computer program which reads the HTML document out loud. The screen reader has no problem saying what b and i mean. However, if you're blind, how do you know what bold or italic means? When we write content, we use bold and italics to signify extra meaning in the text. This meaning is much more easily conveyed when we say strong instead of bold, and emphasis instead of italicized. If you forget this rule, yes, Pachyderm will still show what you mean, but it is important to make sure all users of the presentation are able to grasp the full meaning of the text. Caveat: this isn't exactly the truth, but unless you're going into web development, this is all you need.
